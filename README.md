# Python Port Scanner
 
პროექტის თემა IPv4 TCP Port Scanner

ეს არის მარტივი და ეფექტური პითონზე დაფუძნებული tool რომელიც ასკანირებს მომხარებლის მიერ შეყვანილ IPv4 მისამართებს და ღია პორტის პოვნის შემთხვევაში შემთხვევაში, მონაცემს შეინახავს ტექსტურ ფაილში, შესაბამისი დროის აღრიცხვით.
# ფუნქცია

იმისათვის რომ ქსელის პორტები დავასკანიროთ, საჭიროა ჯერ შევიყვანოთ საწყისი IP მისამართი, შემდეგ საბოლოო IP მისამართი.
ამის შემდეგ უნდა შევიყვანოთ საწყისი პორტი, შემდეგ კი საბოლოო პორტი.

# მოთხოვნები
საჭიროა პითონის ინსტალაცია.
აღნიშნული სკრიპტი იყენებს სტანდარტულ პითონის ბიბლიოთეკებს, ამიტომაც სხვა არაფრის დამატება არ არის საჭირო.
